# Sistema de Gerenciamento de Chaves 🔑 👷 💻
## Descrição:
### Sistema completo de Cadastro de usuários e chaves. O objetivo do projeto é o registro de retirada e entrega de chave, por data, hora e nome de quem retirou. Java (backend robusto e orientado a objetos.MySQL (banco de dados relacional). CRUD e testes unitário com Junit.
👨‍💻
##Tecnologias utilizadas
###Spring Boot (estruturação da aplicação e gerenciamento de dependências)
###Spring MVC (controle de rotas e fluxo da aplicação)
###Spring Data JPA (persistência de dados simplificada)
###Thymeleaf (renderização de páginas dinâmicas no frontend)
